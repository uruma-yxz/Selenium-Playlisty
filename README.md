### Este projeto utiliza Selenium com Python para automatizar a extração e organização de vídeos de playlists do YouTube.

É a ferramenta ideal para quem deseja organizar conteúdos sem precisar estar logado em uma conta do YouTube ou para quem prefere exportar os links dos vídeos em um arquivo .txt para compartilhar facilmente com amigos ou manter um backup offline.

### 🚀 Funcionalidades

    Extração Automática: Coleta links e títulos de vídeos diretamente de uma URL de playlist.

    Organização Prática: Salva as informações de forma estruturada.

    Exportação para TXT: Gera um arquivo de texto pronto para envio ou armazenamento pessoal.

    Privacidade: Não exige login ou integração com a API oficial do Google.

### 🛠️ Tecnologias Utilizadas

Para rodar este projeto, você precisará das seguintes ferramentas:

    Python 3.x: Linguagem base do projeto.

    Selenium: Biblioteca para automação de navegador.

    Webdriver: (Chrome, Firefox ou Edge) compatível com seu navegador.

### 📋 Requisitos e Instalação

    Clone o repositório:
    Bash

    git clone https://github.com/uruma-yxz/Selenium-Playlisty.git

    Instale as dependências:
    Bash

    pip install selenium

    Certifique-se de ter o driver do navegador instalado (Ex: chromedriver para Google Chrome).

### 📖 Como Usar

    Abra o arquivo principal e insira a URL da playlist desejada.

    Execute o script:
    Bash

    python main.py

    O arquivo playlist.txt será gerado automaticamente na raiz do projeto com todos os links organizados.
